AJAX Control Toolkit — September 2009 Release

This release includes two important new controls:

  • SeaDragon Java Script Code (SJC) — The SJC control allows you SeaDragon scripts are used to display an image, and to zoom in and out of that image using mouse button keys without resizing the window.
  • AsyncFileUpload Control — The AsyncFileUpload control enables you to upload files to the server without performing a postback. It supports both client and server progress events.

This release also contains more than 20 bug fixes.

What's in the Download

Open the solution file named AjaxControlToolkit.sln to view the source files for the AJAX Control Toolkit.

The solution contains the following projects:

  • AjaxControlToolkit - The ASP.NET AJAX controls and source
  • SampleWebsite - A website that demonstrates how to use the controls
  • ToolkitTests - A website that contains automated tests for the controls

To get started, simply load the "AjaxControlToolkit.sln" solution in Visual Studio 2008, and run the solution. This will launch the website and allow you to experiment with the different controls. Also included in the solution is the source code for all the controls.
